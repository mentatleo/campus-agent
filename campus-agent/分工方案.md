## 队友A — 前端 + 知识库

**技能要求：零编程。会打字 + 改简单 HTML 数值即可。**

### 一、知识库编写（🔴 P0，Week 1）

在 `data/knowledge/` 下写 txt，每类 5 条 × 每条约 200 字：

| 文件 | 写什么 | 例子 |
|------|--------|------|
| `01-教务管理.txt` | 选课流程、补考重修、转专业辅修 | "选课分三轮：预选→正选→补退选……" |
| `02-奖助学金.txt` | 国奖/励志/助学金/校级 | "国奖8000元/年，综测前5%……" |
| `03-宿舍生活.txt` | 门禁、用电、洗衣热水、网络 | "晚11点门禁，电费0.5元/度……" |
| `04-食堂餐饮.txt` | 各食堂窗口、营业时间、价格 | "一食堂麻辣香锅人均15元……" |
| `05-图书馆.txt` | 借阅规则、开馆、座位预约 | "早8-晚10，借书30天……" |
| `06-校园卡.txt` | 充值、挂失、消费限额 | "微信充值，日消费限额100元……" |
| `07-医疗保障.txt` | 校医院、就诊、报销 | "东校区，门诊8:00-17:30……" |
| `08-就业创业.txt` | 就业中心、招聘会、创业 | "每周三专场招聘会……" |
| `09-社团活动.txt` | 社团、报名、活动场地 | "87个注册社团，百团大战招新……" |
| `10-校园交通.txt` | 校车、共享单车、公交 | "校车每20分钟一班……" |

**50 条约 10000 字，每天 2 类，5 天完成。**

### 二、前端文案修正（🟡 P1）

打开 `frontend/index.html`，找到这些位置改文字：

| 位置 | 找什么 | 改成什么 |
|------|--------|----------|
| 页面标题 | `<title>...</title>` | 确认是"校园数据导航智能体" |
| 欢迎语 | `欢迎使用` | 改成有吸引力的文案 |
| Tab 标签 | `课表查询` `消费分析` 等 | 可加简短说明文字 |
| AI 助手提示 | 默认占位文字 | 加几条引导提问示例 |

不用写代码，搜中文文字替换就行。
