# RAG 知识库内容目录

请在此目录下按类别创建纯文本文件，每个文件包含该类别的知识条目。

## 格式要求

- 每条知识用 `###` 分隔
- 每条 100-300 字
- 纯文本，无需任何标记语言

## 示例格式

```
### 选课流程
衡水学院每学期第16-18周为选课周，学生在教务系统（jwc.hsnc.edu.cn）登录后...
必修课由系统自动置入，选修课需手动勾选...

### 补考规定
期末考试不及格（成绩<60分）的学生可在下学期开学第2周参加补考。
补考成绩最高计为60分。补考仍不及格需重修该课程...
```
